# EiMultas - versão 2 com base documental LGPD

Pacote pronto para GitHub Pages.

## Arquivos incluídos
- index.html
- styles.css
- script.js
- assets/logo.png
- assets/favicon.png
- pages/politica-de-privacidade.html
- pages/termos-de-uso.html
- pages/lgpd.html

## Observação
Este pacote melhora bastante a apresentação institucional relacionada à privacidade, mas não substitui revisão jurídica individualizada da operação.

## Publicação no GitHub Pages
1. Crie um repositório no GitHub
2. Envie todos os arquivos
3. Vá em Settings > Pages
4. Selecione Deploy from a branch
5. Escolha a branch main e a pasta /(root)
6. Salve
